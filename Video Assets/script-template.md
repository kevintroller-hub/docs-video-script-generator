# Video Script Template

This is the template structure for [Your Brand] video scripts, extracted from `video-script-template.pdf`.

## Metadata Section

```
Title:
Short Description:
Org ID:
Username:
Password:
Is 2FA enabled?
Video destination:    Help topic, in-app, devdoc
Slack channel:        #[your-video-channel]
Stakeholders:         List all stakeholders who will be part of the final approval (writers, PMs).
                      All stakeholders should be added to the Slack channel.
Project #:            [Video team will add]
```

## Production Checklist

Production can't start until these steps have been taken:

- Your script has completed the video team's editorial review
- Check your script with the Top 12 Tips for Successful Video Scripts
- Submit your script and an editor will review
- Respond and resolve editorial comments when tagged in the script
- Action On Screen Column has accurate and clear instructions
- Your Test Org is functional and has presentable data
- Final Title and Short Description filled out
- All stakeholders have given feedback on the script

The video team requires an org to film. If this isn't possible, check with the video team for alternative solutions.
